Hibernate Console Application

📌 Project Title

Hibernate Console-Based CRUD Application

📖 Description

This is a Java console application developed using Hibernate ORM to perform basic database operations.

The project demonstrates how to integrate Hibernate with a relational database and execute CRUD (Create, Read, Update, Delete) operations using Hibernate configuration and mapping.

🚀 Features

Add new records to database

View existing records

Update record details

Delete records

Uses Hibernate ORM framework

Console-based user interaction

🛠️ Technologies Used

Java

Hibernate ORM

JDBC

MySQL / Oracle (depending on configuration)

Maven
